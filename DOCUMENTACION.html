🚀 Guía Rápida de Inicio

1

Clonar o Descargar

Obtén los archivos del proyecto desde GitHub o descárgalos directamente.

2

Configurar Valores Personales

Abre script.js y actualiza: nombre, fecha de aniversario, y nombre de canción.

3

Añadir Tu Canción

Coloca tu archivo MP3 en la mismo carpeta que index.html.

4

Probar Localmente

Abre un servidor local o usa python -m http.server 8000 y accede a http://localhost:8000

5

Subir a GitHub Pages

Sube tu proyecto a un repositorio GitHub y activa Pages para publicarlo en línea.

⚙️ Configuración Personalizada

Ubicación del Archivo

Abre script.js y busca esta sección (línea ~9):

const CONFIG = { nombrePersona: "Mi amor", // 🔴 CAMBIA AQUÍ fechaAniversario: "2109", // 🔴 CAMBIA AQUÍ (DDMM) cancion: "audio.mp3", // 🔴 CAMBIA AQUÍ habilitarSeguridad: true // Cambiar a false para sin contraseña };

Ejemplos de Configuración

Ejemplo 1: Sans Valentín

const CONFIG = { nombrePersona: "Mi Valentina", fechaAniversario: "1402", cancion: "love-song.mp3", habilitarSeguridad: true };

Ejemplo 2: Acceso Directo (Sin Contraseña)

const CONFIG = { nombrePersona: "Amor Mío", fechaAniversario: "0101", cancion: "mi-cancion.mp3", habilitarSeguridad: false };

Formatos de Fecha (DDMM)

01/01 → "0101" 14/02 → "1402" ❤️ San Valentín 05/03 → "0503" 15/06 → "1506" 21/09 → "2109" ✨ Ejemplo 24/12 → "2412" 🎄 Navidad

🎵 Agregar Tu Canción

✅ Pasos Correctos:
  1. Descarga una canción en MP3
  2. Colócala en la carpeta "carta" (junto a index.html)
  3. Actualiza el nombre en CONFIG → cancion: "tu-archivo.mp3"
  4. ¡Hecho! Debería reproducirse al abrir la carta
⚠️ Errores Comunes:
  • ❌ Canción en otra carpeta
  • ❌ Nombre mal escrito (respeta mayúsculas)
  • ❌ Archivo dañado o incompleto
  • ❌ Formato no soportado (intenta MP3)

Formatos Soportados

✅ Soportados: .mp3 (recomendado), .wav, .ogg, .m4a ❌ No soportados: .flac, .wma, .aac

🔐 Sistema de Seguridad

¿Cómo funciona?

  1. Usuario abre la página
  2. Aparece un prompt pidiendo código de acceso
  3. Si ingresa la fecha correcta (DDMM) → Acceso concedido ✅
  4. Si es incorrecta → Acceso denegado, puede reintentar ❌

Deshabilitar la Seguridad

// En script.js, línea ~12: habilitarSeguridad: false // Ya no pedirá contraseña

🆘 Solución de Problemas

La música no se reproduce

  1. ✓ Verifica que el archivo esté en la carpeta correcta
  2. ✓ Asegúrate que el nombre exacto coincida en script.js
  3. ✓ Intenta con otro navegador
  4. ✓ Abre la consola (F12) y busca errores

La contraseña no funciona

  1. ✓ Verifica el formato DDMM (4 dígitos, sin caracteres)
  2. ✓ Revisa que sea tu fecha real de aniversario
  3. ✓ Ejemplo: 21 de septiembre = "2109" (no "21-09" o "21/09")

Las partículas no aparecen

  1. ✓ Recarga la página (Ctrl+Shift+R)
  2. ✓ Intenta en otro navegador
  3. ✓ Abre la consola (F12) y busca errores
